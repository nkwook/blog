## 1. 왜 이 글을 쓰는가

Kevin Warsh가 차기 연준(Fed) 의장으로 지명되었다. 그의 핵심 주장은 명확하다:

> "연준 대차대조표가 $7T 이상으로 비대해졌고, $4T 수준으로 축소해야 한다."

이 한 문장을 제대로 이해하려면 몇 가지 질문에 답할 수 있어야 한다.

- 연준 대차대조표가 크면 왜 문제인가?
- 축소하면 시장에 어떤 영향이 있는가?
- 은행주에는 어떤 의미인가?

이 글은 이 질문들에 대한 나의 공부 노트다. Q&A 형식으로 하나씩 풀어본다.

---

## 2. 등장인물: 재무부, 연준, 은행/민간

거시 유동성을 이해하려면 세 플레이어를 구분해야 한다.

| 플레이어 | 역할 | 핵심 계좌 |
|---------|------|----------|
| **재무부 (Treasury)** | 국채 발행, 정부 지출 | TGA (Treasury General Account) |
| **연준 (Fed)** | 통화정책, 국채 매매, 금리 설정 | 준비금(Reserves), SOMA 포트폴리오 |
| **은행/민간** | 예금 보유, 대출, 투자 | 예금(Deposits), 준비금(Reserves) |

**핵심:** 이 세 플레이어 사이에서 돈이 어떻게 이동하느냐가 유동성의 본질이다.

---

## 3. 예금 vs 준비금 — 돈의 두 층위

**Q: 예금과 준비금은 어떻게 다른가?**

돈은 두 개의 층(layer)으로 존재한다:

- **예금(Deposits):** 민간 레이어. 우리가 은행에 가진 돈. 기업이 투자에 쓰는 돈.
- **준비금(Reserves):** 은행 시스템 레이어. 은행들이 연준에 가진 돈. 은행 간 결제에 사용.

```
┌─────────────────────────────────┐
│       민간 레이어 (예금)          │  ← 우리가 보는 돈
├─────────────────────────────────┤
│    은행 시스템 레이어 (준비금)      │  ← 은행 간 결제용
├─────────────────────────────────┤
│         연준 (중앙은행)           │  ← 준비금의 원천
└─────────────────────────────────┘
```

**포인트:** 연준이 QE로 "돈을 푼다"는 것은 준비금을 만드는 것이다. 이것이 자동으로 민간 예금이 되는 건 아니다. 하지만 은행 시스템의 유동성이 풍부해지면 금융 조건이 완화되고, 이것이 결국 시장과 실물경제에 영향을 미친다.

---

## 4. 국채 발행 → 유동성 흡수

**Q: 재무부가 국채를 발행하면 유동성은 어떻게 되나?**

국채 발행의 자금 흐름:

```
민간 → 국채 매입 → 예금 감소(↓)
                 → 은행 준비금 감소(↓)
                 → TGA 잔고 증가(↑)
```

1. 민간이 국채를 산다 → 민간 예금이 줄어든다
2. 은행은 연준에서 준비금을 TGA로 이체한다
3. TGA(정부 금고)에 돈이 쌓인다

**결과:** 국채 발행 = 민간과 은행 시스템에서 유동성을 **흡수**하는 행위

---

## 5. TGA: 유동성 잠금/해제 장치

**Q: TGA에 쌓인 돈은 어떻게 되나?**

TGA(Treasury General Account)는 재무부가 연준에 갖고 있는 계좌다.

- **TGA 잔고 증가 = 유동성 잠금(Lock):** 민간에서 빠져나온 돈이 정부 금고에 갇힌다
- **TGA 잔고 감소 = 유동성 해제(Unlock):** 정부가 지출하면 돈이 민간으로 돌아온다

```
국채 발행          정부 지출
민간 → TGA(↑)     TGA(↓) → 민간
유동성 흡수         유동성 방출
```

**포인트:** 정부 지출은 유동성 방출이다. 재무부가 TGA에 돈을 쌓아두면 시장 유동성이 줄고, 지출하면 늘어난다. 부채한도 협상 같은 이벤트로 TGA가 급감하면 시장에 유동성이 쏟아지는 이유가 이것이다.

---

## 6. QE: 연준의 준비금 생성 구조

**Q: QE(양적완화)는 구체적으로 어떻게 작동하나?**

연준이 국채나 MBS를 매입하는 과정:

```
연준: 국채 매입
  → 은행에 준비금 생성(↑)      [은행 시스템 레이어]
  → 민간에 예금 생성(↑)        [민간 레이어, 매도자가 비은행인 경우]
  → 연준 자산 증가(↑)         [SOMA 포트폴리오]
```

**핵심 메커니즘:**

1. **준비금 창출:** 연준이 국채를 사면 그 대금을 "새로 만든" 준비금으로 지급한다. 이것이 "돈을 찍는다"의 실체다.
2. **포트폴리오 리밸런싱:** 국채를 판 투자자는 그 돈으로 더 위험한 자산(회사채, 주식 등)을 산다 → 자산 가격 상승 → 금융 조건 완화
3. **금리 하방 압력:** 연준의 대량 매입 → 국채 가격 상승 → 금리 하락

**QT(양적긴축)는 역과정:**

```
연준: 만기 도래 국채 재투자 안 함
  → 은행 준비금 감소(↓)
  → 연준 자산 감소(↓)
```

---

## 7. 금리 인상과 변동금리 전환

**Q: 금리 인상이 연준 자체에 미치는 영향은?**

여기가 많은 사람이 놓치는 부분이다.

연준은 은행의 준비금에 **이자(IORB, Interest on Reserve Balances)**를 지급한다. QE로 준비금이 $4T 이상 쌓인 상태에서 금리를 5%대로 올리면:

```
연준 이자 비용 = 준비금 규모 × IORB 금리
             = ~$3.5T × ~5.4%
             = 연간 ~$1,900억
```

반면 연준이 보유한 국채/MBS는 대부분 저금리 시절에 매입한 것이라 수익은 낮다.

**결과:**
- 연준이 **적자(순손실)**를 기록
- 재무부에 대한 송금(remittance)이 **0원**으로 감소
- 이것은 사실상 정부 재정에 부담 = 납세자 비용

**Warsh의 핵심 비판 포인트가 여기다:** 비대해진 대차대조표 + 높은 금리 = 연준 스스로가 재정 부담의 원인이 됨.

## 8. 유동성 스위치 총정리

| 스위치 | ON (유동성 증가) | OFF (유동성 감소) |
|-------|-----------------|-----------------|
| **QE/QT** | QE: 연준 자산 매입 → 준비금↑ | QT: 만기 국채 미재투자 → 준비금↓ |
| **TGA** | TGA 감소(정부 지출) → 민간↑ | TGA 증가(국채 발행) → 민간↓ |
| **RRP** | RRP 감소 → 준비금↑ | RRP 증가 → 준비금↓ |
| **금리** | 금리 인하 → 금융조건 완화 | 금리 인상 → 금융조건 긴축 |

**RRP(Reverse Repo Facility) 보충 설명:**
- MMF(머니마켓펀드) 등이 연준에 현금을 맡기고 국채를 담보로 받는 제도
- RRP 잔고가 높으면 시장에서 유동성이 빠져나간 상태
- 2022~2024년 RRP 감소 = 시장에 유동성이 다시 풀린 주요 원인 중 하나

## 9. Kevin Warsh의 주장을 다시 보면

이제 Warsh의 주장을 구체적으로 분해할 수 있다:

### (1) 대차대조표 $7T → $4T 축소

- 현재 연준 보유 자산: ~$6.8T (국채 ~$4.2T + MBS ~$2.2T + 기타)
- 목표: $4T 수준으로 약 $3T 축소
- 이는 단순히 만기 도래 미재투자(passive QT)로는 수년이 걸리는 규모

### (2) MBS 적극 매각

- 기존 QT: 만기 도래 국채만 미재투자 (수동적)
- Warsh: MBS를 **적극적으로 시장에 매각** (능동적)
- 연준이 $2T+ MBS를 시장에 내놓으면 → MBS 스프레드 확대 → 모기지 금리 상승

### (3) "QT + 금리인하" 동시 추진

- 전통적으로 QT와 금리인하는 모순처럼 보임
- Warsh의 논리: "대차대조표 정상화(QT)는 구조적 과제, 금리는 경기순환적 도구 — 둘은 독립적"
- 금리를 내려 경기를 지지하면서, 동시에 대차대조표를 줄여 시장의 가격발견 기능을 복원

### (4) 시장의 가격발견 기능 복원

- QE 기간 동안 연준이 국채/MBS의 최대 매수자 → 시장 가격이 왜곡
- 연준이 빠지면 → 시장 참여자들이 자체적으로 리스크를 평가하고 가격을 결정
- 이것이 Warsh가 말하는 "Privatize QE" — 중앙은행이 아닌 민간이 자본 배분을 주도

## 10. 은행주 투자 내러티브로의 연결

연준 대차대조표 메커니즘을 이해하면, 단순히 "금리 오르면 은행주 좋다"를 넘어 더 입체적인 판단이 가능하다.

### 수혜 섹터 — 대형은행 (JPM, BAC, C, GS, WFC)

**Yield Curve 스티프닝 → NIM 확대:**
- Warsh 체제에서 단기 금리 인하 + 장기 금리는 시장이 결정(QT로 연준 매수 감소)
- 장단기 금리차(yield curve) 확대 → 은행의 순이자마진(NIM) 개선
- 은행의 핵심 수익 구조: 단기 예금 조달 → 장기 대출/국채 투자

**규제 완화 기대:**
- 트럼프 행정부 + Warsh 조합 → 자본요건 경감 기대 (Basel III Endgame 완화)
- 자본요건 완화 → 자사주 매입·배당 여력 확대
- 대형은행의 ROE(자기자본이익률) 상승 요인

**"Privatize QE" = 대형 딜러 은행에 유리:**
- 연준이 시장에서 빠지면 → 민간 금융기관이 국채/MBS 거래의 주축
- 거래량 증가 → 딜러 은행의 트레이딩 수익 확대
- Goldman Sachs, JPMorgan 등 FICC(채권·통화·상품) 부문 수혜

### 리스크 섹터 — 모기지/REIT

**MBS 스프레드 확대:**
- 연준의 $2T+ MBS 적극 매각 → MBS 공급 급증
- MBS 스프레드 확대 → AGNC, NLY 등 모기지 REIT의 포트폴리오 가치 하락 + 펀딩 비용 상승
- 레버리지가 높은 모기지 REIT일수록 타격 큼

**리파이낸싱 볼륨 감소:**
- MBS 매각 → 모기지 금리 상승 → 리파이낸싱 수요 감소
- Rocket Companies, UWM Holdings 등 모기지 렌더 매출 압박

### 핵심 포인트

| 구분 | 메커니즘 | 대표 종목 |
|-----|---------|----------|
| 수혜 | Yield curve 스티프닝 → NIM 확대 | JPM, BAC, WFC |
| 수혜 | 규제 완화 → 자본 환원 확대 | C, GS |
| 수혜 | Privatize QE → 트레이딩 수익 | GS, JPM |
| 리스크 | MBS 매각 → 스프레드 확대 | AGNC, NLY |
| 리스크 | 모기지 금리 상승 → 볼륨 감소 | RKT, UWMC |

**결론:** 연준 대차대조표 축소를 단순히 "긴축"으로만 보면 안 된다. Warsh 체제에서는 **대차대조표 축소 + 금리 인하 + 규제 완화**가 패키지로 올 수 있고, 이 조합은 대형은행에게 상당히 우호적인 환경이다. 반면 모기지 REIT와 모기지 렌더는 구조적 역풍을 맞는다.
